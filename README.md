# 3D-Timelapse-Viewer-of-Satellite-Image-Data
A real-time 3D renderer of satellite image data using Landsat data for RGB and thermal values and ASTER data for elevation.

Features: 
* Timelapse Animation
* Tracking of geographical features: 
  * Surface water area 
  * Snow coverage 
  * Vegetation spread
